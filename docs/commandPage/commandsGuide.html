Schedule 1 Guide

Home Mixing Guide Commands Guide

freecam

Toggles free cam mode.

Exemple : freecam

save

Forces a save.

Exemple : save

settime

Sets the time of day to the specified 24-hour time.

Exemple : settime 1530

give

Gives the player the specified item. Optionally specify a quantity.

Exemple : give ogkush 5

clearinventory

Clears the player's inventory.

Exemple : clearinventory

changecash

Changes the player's cash balance by the specified amount.

Exemple : changecash 5000

changebalance

Changes the player's online balance by the specified amount.

Exemple : changebalance 5000

addxp

Adds the specified amount of experience points.

Exemple : addxp 100

spawnvehicle

Spawns a vehicle at the player's location.

Exemple : spawnvehicle [vehicle name]

setmovespeed

Sets the player's move speed multiplier.

Exemple : setmovespeed 1

setjumpforce

Sets the player's jump force multiplier.

Exemple : setjumpforce 1

teleport

Teleports the player to the specified location.

Exemple : teleport [location]

setowned

Sets the specified property or business as owned.

Exemple : setowned [property/business]

packageproduct

Packages the equipped product with the specified packaging.

Exemple : packageproduct [packaging]

setstaminareserve

Sets the players stamia reserve (default 100) to the specified amount.

Exemple : setstaminareserve 200

raisewanted

Raises the player's wanted level.

Exemple : raisewanted

lowerwanted

Lowers the player's wanted level.

Exemple : lowerwanted

clearwanted

Clears the player's wanted level.

Exemple : clearwanted

sethealth

Sets the player's health to the specified amount.

Exemple : sethealth 100

settimescale

Sets the time scale. Default 1.

Exemple : settimescale 1

setvar

Sets the value of the specified variable.

Exemple : setvar [variable] [value]

setqueststate

Sets the state of the specified quest entry.

Exemple : setqueststatec [quest name] [entry index]

setquestentrystate

Sets the state of the specified quest entry.

Exemple : setquestentrystate [questname] [entry index] [state]

setemotion

Sets the facial expression of the player's avatar.

Exemple : setemotion [emotion]

setunlocked

Unlocks the given NPC.

Exemple : setunlocked [npc_id]

setrelationship

Sets the relationship scalar of the given NPC. Range is 0 to 5.

Exemple : setrelationship [npc_id] 5

addemployee

Adds an emoloyee of the specified type to the given property.

Exemple : addemployee [employee title] [property]

setdiscovered

Sets the specified product as discovered.

Exemple : setdiscovered [product]

growplants

Sets ALL plants in the world fully grown.

Exemple : growplants

setlawintensity

Sets the intensity of law enforcement activity on a scale of 0 to 10.

Exemple : setlawintensity 6

setquality

Sets the quality of the currently equipped item.s

Exemple : setquality [quality]

bind

Binds the given key to the given command.

Exemple : bind [key] [command]

unbind

Removes the given bind.

Exemple : unbind [key]

clearbinds

Clear ALL binds.

Exemple : clearbinds

hideui

Hides all on-screen UI.

Exemple : hideui

disable

Disables the specified GameObject.

Exemple : disable [GameObject]

enable

Enables the specified GameObject.

Exemple : enable [GameObject]

endtutorial

Forces the tutorial to end immediately if the player is actually in the tutorial.

Exemple : endtutorial

disablenpcasset

Disabled the given asset under all NPCs.

Exemple : disablenpcasset [asset]

showfps

Shows FPS label on the top left.

Exemple : showfps

hidefps

Hides FPS label.

Exemple : hidefps

cleartrash

Instantly removes all trash from the world.

Exemple : cleartrash
